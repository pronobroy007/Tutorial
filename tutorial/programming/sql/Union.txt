The SQL UNION Operator:
-----------------------
    The UNION operator is used to combine the result-set of two or more SELECT statements.

    * Each SELECT statement within UNION must have the same number of The columns must also 
    * have similar data typescolumns The columns in each SELECT statement must also be in the same order
    Example:
    --------
        SELECT d1, d2, d3 FORM t1
        UNION
        SELECT d1, d2, d3 FORM t2

        >> Here in two select statement we have same number of column and order also same


    Note: UNION selects only distinct values. Use UNION ALL to also select duplicate values!
          UNION is slower than UNION ALL because UNION make a distinct sort.

Example:
--------
    SELECT City, Country FROM Customers
    WHERE Country='Germany'
    UNION ALL
    SELECT City, Country FROM Suppliers
    WHERE Country='Germany'
    ORDER BY City;

    >> SQL statement returns the German cities (duplicate values also) 
        from both the "Customers" and the "Suppliers" table
